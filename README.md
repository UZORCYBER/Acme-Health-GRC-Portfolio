# Acme Health GRC Portfolio

## Overview

This project simulates a real-world Governance, Risk, and Compliance (GRC) consulting engagement for a fictional healthcare company.

The goal is to assess the organization's cybersecurity posture, identify risks, evaluate compliance against major security frameworks, and recommend improvements.

This portfolio demonstrates the work expected from an Entry-Level GRC Analyst.

---

## Objectives

- Perform enterprise risk assessments
- Build and maintain a risk register
- Develop cybersecurity policies
- Conduct a NIST CSF assessment
- Complete an ISO 27001 gap assessment
- Evaluate SOC 2 readiness
- Crosswalk multiple compliance frameworks
- Create executive-level security reports

---

## Frameworks

- NIST Cybersecurity Framework (CSF)
- ISO/IEC 27001
